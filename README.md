# April Hackathon – Mini E‑Commerce Backend

This project is a FastAPI-based backend built as part of the April Hackathon.

- User management (CRUD)
- Order management
- SQLite database
- Pytest test suite

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- Pytest
